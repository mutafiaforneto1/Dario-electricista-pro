# Comprehensive Guide for Using the Telegram Bot

## Introduction
This guide provides detailed instructions on how to use the Telegram bot in the **Dario-electricista-pro** repository.

## Commands
The following commands are available for interacting with the bot:

### /nuevo
- **Description:** Initiates the process to create a new request.
- **Usage:** Simply send the command `/nuevo` to the bot. Follow the prompts to complete your request.

### /resumen
- **Description:** Provides a summary of your current requests or interactions.
- **Usage:** Send the command `/resumen` to the bot to receive a concise overview of your activities.

### /cancel
- **Description:** Cancels the current operation or request.
- **Usage:** If you want to halt an ongoing process, type `/cancel` and the bot will stop the current action.

### /editar
- **Description:** Allows you to edit an existing request.
- **Usage:** To edit a request, send `/editar` followed by the details of what you wish to modify. The bot will guide you through the editing process.

### /factura
- **Description:** Generates and sends an invoice based on your current transactions.
- **Usage:** By sending `/factura`, you will receive an invoice formatted according to your latest requests.

### /contacto
- **Description:** Provides contact information for support or further inquiries.
- **Usage:** Use the command `/contacto` to ask for contact details. The bot will provide you with the relevant information.

### /metrics
- **Description:** Displays metrics related to your usage or performance statistics.
- **Usage:** To view metrics, send the command `/metrics`, and the bot will return a summary of useful statistics.

## Conclusion
This guide aims to ensure that you can effectively use the Telegram bot in the Dario-electricista-pro repository, enhancing your experience and productivity.
